# Minigames With Friends!
Minigames With Friends is a unique minecraft plugin that bundles together 6 different fun minigames that you can play either separately, or mixed together for an even better experience!

## Minigames included:
- **Blockshuffle:** All players will be assigned a block randomly, and stepping on that block before time runs out awards players points.

- **DeathSwap:** Every x minutes players will swap locations with eachother. Players earn points by killing the person who swapped with them, for example, by having them fall to their death.
- **Randomizer:** All blocks now drop a random, completly different item than they're supposed to. Players must now attempt to find out how to get good gear with this restriction. [No points are gained from this gamemode].

- **DeathMatch:** Every x minutes, players will duke it out to the death in an arena, preserving their inventory on death. Last player standing earns a point.
- **DimensionSwap:** A totally new, never seen before gamemode! Every x minutes players will change the dimension they're in, into any custom minecraft map from a preconfigured pool. Every x swaps, players will have a deathmatch, with the last player surviving earning a point. [Additional setup required]
- **Would You Rather:** A unique minigame heavily inspired by EightSidedSquare's WouldYouRather mod. Every x minutes players will be prompted to choose between 2 options, with both options containing a good and a bad effect. Players attempt to create a build that is most benefical to them, to eventually help them win the other minigames. [No points are gained from this gamemode] [Also available as a standalone plugin](https://modrinth.com/project/would-you-rather-plugin), due to its size and complexity.

## Getting Started
To get Minigames With Friends up and running, here and the steps you need to follow.
1. Install the plugin on your server
2. (Optional) Configure the settings for each gamemode. This can be done by using the command `/mg config <gamemode name> [config key] [config value]` The tab completions will be quite helpful in showing you the avaible gamemodes and config options. For more about plugin configuration, look at the configuration guide [HERE](https://example.com)
3. Choose and enable what gamemodes you want to play. If a game is not currently in progress, enabling a gamemode will not start it immediately, but rather queue it up for when the game does start. This can be done by doing `/mg EnableGamemode <gamemode name>`. It's also totally supported (and intended) that you enable multiple gamemodes at the same time, and while any combination will work with eachother, you can find a list of recommended combinations [HERE](https://example.com).
4. Start the game by doing `/mg start`. This will start the game, will all enabled gamemodes now going into effect.
